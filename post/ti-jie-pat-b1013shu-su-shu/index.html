题解PAT B1013:数素数

2019-06-20

题目：

https://pintia.cn/problem-sets/994805260223102976/problems/994805309963354112

思路:

素数筛法 + 打表：用筛法（筛法的详细讲解见胡凡《算法笔记》P162）求出范围内第1-第10^4个素数，根据要求输出。

如何知道第10^4个素数大约是多少？需要先调整定义过的maxn的范围，输出FindPrime()的cnt值来获得。经测试maxn==100000筛出9593个素数，maxn==150000筛出13849个素数，所以maxn选择150000即可。

题解:

#include <iostream>
using namespace std;
const int maxn = 200500;        //要大于第10^4个素数的范围
int prime[maxn] = {0};
int cnt = 1;
void FindPrime(){           //素数筛
    bool a[maxn] = {false};
    for (int i = 2; i < maxn; ++i) {
        if (a[i] == false){
            for (int j = i + i ; j < maxn ; j = j + i) {
                a[j] = true;
            }
            prime[cnt] = i;
            cnt++;
        }
    }
}
int main(){
    int ans = 1;
    int s,d;
    cin>>s>>d;
    FindPrime();
    for (int i = s; i <= d; ++i,ans++) {
        if (ans % 10 == 0){	//每10个数一换行+去掉空格
            cout<<prime[i];
            cout<<endl;
        }else if(i != d ){
            cout<<prime[i]<<" ";
        }else if(i == d ){	//最后一个输出结果也不能有空格
            cout<<prime[i];
        }
    }
    return 0;
}

What I learned（遇到的坑）

1.maxn一开始开的太小了，有几个测试点WA了
2.输出格式要注意，最后一个输出结果后面也不能带有空格，否则就会Presentation Error
