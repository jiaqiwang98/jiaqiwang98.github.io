题解Codeup6126:Set Similarity

2019-06-16

题目：

本题为胡凡《算法笔记》STL章节的一道练习题。
http://codeup.cn/problem.php?cid=100000597&pid=0

题目大意：

给定几个集合，求指定编号集合的相似率。

相似率为：交集的元素个数/并集的元素个数*100%。

题解：

#include <iostream>
#include <set>
#include <iomanip>
using namespace std;
int main(){
    set<int> s[60];	
    int n;
    cin>>n;
    int a,pos = 1,t;
    while(n--){
        cin>>a;
        for (int i = 0; i < a; ++i)
        {
            cin>>t;
            s[pos].insert(t);
        }
        pos++;
    }
    int b,l,r;
    double rate;
    cin>>b;
    while (b--){
        cin>>l>>r;
        int cnt = 0;
        set<int>ss;
        set<int>::iterator it;
        for (it = s[l].begin(); it != s[l].end(); it++) {
            ss.insert(*it);
            if (s[r].find(*it) != s[r].end()){
                cnt++;
            }
        }
        for (it = s[r].begin(); it != s[r].end(); it++) {
            ss.insert(*it);
        }
        rate = cnt * 1.0 / ss.size() * 100;
        printf("%.1lf%%\n",rate);
    }
    return 0;
}

What I Learned（遇到的坑）

1.set<int> s[60] 容器类器也可以定义为数组

2.find()函数如果没有找到元素则返回s.end()

类型为迭代器而不是布尔值，,所以判断语句应该为if (s[r].find(*it) != s[r].end())

而不是if (s[r].find(*it))
