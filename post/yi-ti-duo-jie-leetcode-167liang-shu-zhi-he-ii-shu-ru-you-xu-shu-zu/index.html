一题多解LeetCode: 167.两数之和 II - 输入有序数组

2019-11-09

本篇作为上一篇一题多解LeetCode:1.两数之和的续篇，与1题条件唯一不同点在于数组有序，主要讲一下双指针(two pointers)的思想。

题目：

给定一个已按照升序排列 的有序数组，找到两个数使得它们相加之和等于目标数。

函数应该返回这两个下标值 index1 和 index2，其中 index1 必须小于 index2。

说明:

返回的下标值（index1 和 index2）不是从零开始的。
你可以假设每个输入只对应唯一的答案，而且你不可以重复使用相同的元素。

示例：

输入: numbers = [2, 7, 11, 15], target = 9
输出: [1,2]
解释: 2 与 7 之和等于目标数 9 。因此 index1 = 1, index2 = 2 。

基本解法：

做完1.两数之和后应该对暴力和哈希表的做法很熟练了，这两种方法解这道题是没有问题的。
暴力就不写了，放一下哈希表AC的代码。

class Solution {
public:
    vector<int> twoSum(vector<int>& numbers, int target) {
        map<int,int> hash;
        vector<int> res;
        for(int i=0;i<numbers.size();i++){
            hash.insert(map<int,int>::value_type(numbers[i],i));
        }
        for(int i=0;i<numbers.size();i++){
            int temp = target - numbers[i];
            if(hash.find(temp) != hash.end() && hash[temp] != i){
                //因为题目要求index1必须小于index2，所以加个判断
                if(i + 1 > hash[temp] + 1){ 
                    res.push_back(hash[temp]+1);
                    res.push_back(i+1);
                }else{
                //注意题目说明:下标从1开始，元素+1
                    res.push_back(i+1);
                    res.push_back(hash[temp]+1);
                }
                break;
            }
        }
        return res;
    }
};

双指针(two pointers)解法

双指针（有时叫快慢指针）法在LeetCode中使用极其普遍，其最明显的特征就是两个指针同时扫描，从而减少一重循环的工作量，将O(n2)O(n^2)O(n2)的时间复杂度降至O(n)O(n)O(n)。如果你在一道题中需要使用两次遍历解决问题，那么该题往往就能采用双指针的方法优化至一次遍历。
